# HERENCIA EN JAVA
La herencia en Java es un mecanismo que permite que una clase adquiera las propiedades y métodos de otra clase, conocida como clase padre o superclase. La clase hija o subclase hereda los atributos y comportamientos de la clase padre, y puede agregar nuevos atributos y métodos, o modificar los existentes. Esto fomenta la reutilización de código y permite la creación de jerarquías de clases.
